# Fundamentos-da-Reserva-de-Emergencia

Projeto: Caderno Temático com IA (NotebookLM)
Descrição
Este projeto demonstra a utilização da ferramenta NotebookLM para análise de dados financeiros. O objetivo foi testar a capacidade da IA em processar fontes abertas e gerar respostas baseadas em evidências.

Tecnologias Utilizadas
NotebookLM (Google) para processamento de linguagem natural.

Markdown para documentação.

Aprendizados
Engenharia de prompts para extração de dados específicos.

Organização de fontes bibliográficas em ambiente de IA.
